# node-red-puppeteer


### These node-red nodes expose puppeteer's API used for controlling Chrome/Chromium over the <a href='https://chromedevtools.github.io/devtools-protocol/'>DevTools Protocol</a>


Group | Node Name | Detailed information
------------- | ------------- | -------------
<b>Browser</b> | Launch 
<b>Browser</b> | New Page 
<b>Browser</b> | Close Browser 
<b>Page</b> | Click 
<b>Page</b> | Close 
<b>Page</b> | Content 
<b>Page</b> | Delete Cookies 
<b>Page</b> | Download 
<b>Page</b> | Focus 
<b>Page</b> | Get 
<b>Page</b> | Get Cookies 
<b>Page</b> | Go to 
<b>Page</b> | Screenshot 
<b>Page</b> | Viewport
<b>Page</b> | Set 
<b>Page</b> | Set LocalStorage
<b>Page</b> | Set Cookies 
<b>Page</b> | Upload 
<b>Page</b> | Wait for 
<b>Keyboard | Press 
<b>Keyboard | Type 
<b>Storage</b> | Rename 

#### Install 
```bash
npm install @robinck/node-red-puppeteer
```
